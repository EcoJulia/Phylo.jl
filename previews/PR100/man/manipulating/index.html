Phylo.droptips! — Function
droptips!(tree::AbstractTree{OneTree}, tips; keep = false)

Function to drop tips from a phylogenetic tree tree, which are found in the vector of tips or tip names, tips. keep determines whether to keep internal and root nodes that now only have one child (default is false). Internal nodes with no children will always be removed.

source
Phylo.keeptips! — Function
keeptips!(tree::AbstractTree{OneTree}, tips; keep = false)

Function to keep only the tips in a phylogenetic tree, tree, that are found in the vector of tips or tip names, tips. keep determines whether to keep internal and root nodes that now only have one child (default is false). Internal nodes with no children will always be removed.
