Traversing and iterating over trees

The code also provides iterators, and filtered iterators over the branches, nodes, branchnames and nodenames of a tree (using the random tree from Creating phylogenies)

using Phylo
nu = Nonultrametric(5);
tree = rand(nu)

collect(nodeiter(tree))
9-element Vector{LinkNode{OneRoot, String, Dict{String, Any}, LinkBranch{OneRoot, String, Dict{String, Any}, Float64}}}:
 LinkNode tip 1, a tip of the tree with an incoming connection (branch 10).
 LinkNode tip 2, a tip of the tree with an incoming connection (branch 12).
 LinkNode tip 3, a tip of the tree with an incoming connection (branch 15).
 LinkNode tip 4, a tip of the tree with an incoming connection (branch 14).
 LinkNode tip 5, a tip of the tree with an incoming connection (branch 9).
 LinkNode Node 6, an internal node with 1 inbound and 2 outbound connections (branches 11 and 9, 10)
 LinkNode Node 7, an internal node with 1 inbound and 2 outbound connections (branches 13 and 11, 12)
 LinkNode Node 8, an internal node with 1 inbound and 2 outbound connections (branches 16 and 13, 14)
 LinkNode Node 9, a root node with 2 outbound connections (branches 15, 16)
collect(nodenamefilter(isroot, tree))
1-element Vector{String}:
 "Node 9"
